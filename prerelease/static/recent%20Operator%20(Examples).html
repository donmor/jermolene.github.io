recent Operator (Examples)

[recent[1]]
→ tiddlers modified yesterday and today

[recent:created[14]]
→ tiddlers created within the last 14 days

The filter can be used to highlight new items in a list. For example: 


<ul>
  <$list filter="[tag[ReleaseNotes]!<currentTiddler>!sort[modified]]">
    <li>
      <$link><$view field="title"/></$link><$list filter="[<currentTiddler>recent[90]]"> @@color:red;^^new^^@@</$list>
    </li>
  </$list>
</ul>

That renders as:

  • Release 5.1.10 new
  • Release 5.1.9 new
  • Release 5.1.8
  • Release 5.1.3
  • Release 5.1.5
  • Release 5.1.6
  • Release 5.1.7
  • Release 5.1.4
  • Release 5.1.2
  • Release 5.1.1
  • Release 5.1.0
