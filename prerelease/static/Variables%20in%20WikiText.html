Variables in WikiText

31 days ago

Variables contain values. They are available within the scope of the widget in which they are defined, as well as all child widgets in the widget tree.

Defining Variables

Variables are defined by...

Macros
in fact, variables are macro shorthands (see example below)
SetWidget
sets a variable to a value
ListWidget
sets list-item and iterator variabes
TiddlyWiki
in the core and core templates

Using Variables

Variables are used...

via shorthand
<<variableName>>
via variable substitution within macros
$(variableName)$
as parameters to filter steps
[operator<variableName>]
in widgets
internally, or as widget attributes
<$edit tiddler=<<currentTiddler>>/>

Examples

defining a variable via SetWidget

<$set name=foo value=bar>
<<foo>>
</$set>
