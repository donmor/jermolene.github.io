Defining Variables

Variables contains values defined by Widgets. Variables are available within the widget that defines them, and the child widgets in the widget tree.

Variables are defined by:

  • TiddlyWiki core and viewtemplate
  • SetWidget
  • ListWidget
  • Macro definition

Using Variables

Variables are used in:

  • Variable substitution $(name)$
  • Concise shortcut <<name>>
  • Filter expression [operator<variable-operand>]
  • Some default behaviors of Widgets

See also currentTiddler variable, built-in variables and Confusion between Transclusion and Substitution.
