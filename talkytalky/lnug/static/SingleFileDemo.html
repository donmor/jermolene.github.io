Demo of Single File Edition

How an end user might use TiddlyWiki

  1. Visit http://tiddlywiki.com/empty.html in Chrome
  2. Copy and paste part of an HTML page
  3. Drag an image from the file system
  4. Drag task management tiddler from http://tiddlywiki.com and add tasks
  5. Save using Chrome
  6. Inspect source
  7. Reload in Firefox, use TiddlyFox
  8. Enable encryption, resave, check source
  9. Apps for smartphones!
