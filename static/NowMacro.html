NowMacro

15 days ago

The 'now' macro returns the current date and time, formatted with an optional format string.

Parameters

PositionNameDescriptionDefault
1stformatDateFormat string specifying the format for the date/time0hh:0mm, DDth MMM YYYY

Examples

For example:

* <<now>>
* <<now "DDth MMM YYYY">>

Returns:

  • 14:39, 29th October 2014
  • 29th October 2014
